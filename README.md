## 페이지 구성 목록

- User : 사용자 정보 조회
    - 로그인 페이지
        - 정보 수정
    - 회원가입 페이지
        - 비밀번호 일치
        - 이메일 검증
    
- Post : 게시글 페이지
    - 게시글 작성/조회/수정/삭제 

- Comment : 댓글 페이지
    - 댓글 작성/조회/수정/삭제 
    
- Report : 신고 페이지
    - 게시글 신고
    - 댓글 신고
- file : 사진 파일 페이지
- Borad : 게시판 페이지
