## travel-board


## 🏝️ 제주도 여행 게시판 (제주 이야기)

## 주요 기능 및 서비스

- 지역별 게시판 구분
- 로그인 기능 → 필터
- 댓글 기능, 신고 기능
- 이미지 첨부

## 페이지 구성 목록

- User : 사용자 정보 조회
    - 로그인 페이지
        - 정보 수정
    - 회원가입 페이지
    
- Post : 게시판 페이지
    - 게시물 조회
    - 게시물 작성

- Comment : 댓글 페이지
- Report : 신고 페이지
- file : 사진 파일 페이지
- 관리자 페이지
    - 게시글, 작성자 신고
